# BMW 3D Cinematic Showcase

A premium, production-ready BMW website built with Next.js 15, React Three Fiber, Three.js, GSAP ScrollTrigger, and Framer Motion.

## 🚀 Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## 📁 Project Structure

```
bmw-showcase/
├── src/
│   ├── app/          # Layout, pages, global styles
│   ├── components/   # 3D, animations, UI, effects
│   └── lib/          # Data models, utilities
└── public/           # Models, videos, images
```

## 🛠️ Built With

- Next.js 15 + React 19 + TypeScript
- React Three Fiber + Drei (3D rendering)
- GSAP ScrollTrigger (scroll animations)
- Framer Motion (UI animations)
- Tailwind CSS v4 (styling)

## 📥 Add Assets

Place 3D models in `public/models/` and videos in `public/videos/`.
